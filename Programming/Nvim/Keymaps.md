#neovim
##### **Neotree** 
(File system browser)-> ``Ctrl + n``
##### **Telescope**: 
- File search: ``Ctrl + p``
- File grep:     ``Space + f + g``
##### **LSP**
- Info about code: ``Shift + k``
- Go to definition of function: ``g + d`` 
- Code actions: ``Space + c + a`` 
##### **Toggleterm**
- Open terminal ``Ctrl + t``
